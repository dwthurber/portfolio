MSHE Student Resources Site

  • 2014-2015
  • Web Design, ELearning

Project Info

The MS in Higher Education Administration and Policy Program had little to no online community and the existing student and alumni resources site was poorly organized and under-used. With several colleagues, we redesigned the student experience by first moving the existing student and alumni resources site into the learning management system--Canvas--used by Northwestern. This was done because it was a central location that all students use on a daily basis and was thus a convenient place for students to access content and information.

The first major part of this process was a complete overhall of a cluttered and out of use student and alumni resources site. Using Jquery and clever web design, I built a new single page site that was embedable in the Canvas community courses an which allowed for better display and organization of content. Among other things, this project resulted in a web app which could:

  • Be sorted and filtered dynamically.
  • Be searched for content using a jquery search plugin.
  • Redesigned content interface and organization of content.

This project was particularly challenging in figuring out a way to display a wide range of content to students in a way that was easy to organize and find. I learned how to embed a Jquery plugin into the site which allowed for complex filtering of display content. This, along with a reorganization of the content, a completely redesigned (more modern) look, and a custom built search bar created a website which provided easy-to-find information to students and newly admitted students. I also focused heavily on this project on mobile responsive design as many students accessed Canvas through the mobile app. As such, this project greatly strengthed my responsive design skills and understanding of CSS @media queries. This site was published through gitHub as an SSH secured site and made editable to staff members through integration with a static website hosting and editing platform with an embeded WYSIWYG markup editor for non-technical users.
