MSHE Student Resources Site

  • 2014-2015
  • Web Design, ELearning

Project Info

When I came to the Master of Science in Higher Education program office as a graduate assistant, the program was in need of a redesign for how an online community was built and content was displayed and disseminated to students and alumni. The first major part of this process was a complete overhall of a cluttered and out of use student resources site. Using Jquery and clever web design, I built a new site that was embedable in Canvas courses which allowed for better display and organization of content. Among other things, this project resulted in a web app which could:

  • Be sorted and filtered dynamically.
  • Be searched for content using a jquery search plugin.
  • Redesigned content interface and organization of content.
